# Foolish organic chemistry
用python实现对简单有机物的存储,比较
## 操作示例
运行oc_console.py文件以启用控制台
```sh
nm test         # 新建一个分子 分子名称为test
a c 7           # 在分子中添加七个碳原子
a o             # 在分子中添加一个氧原子
list            # 显示当前分子中各原子的编号及状态
c 1,2,3,4,5,6 t     # 将编号为1,2,3,4,5,6的原子用单键顺序相连 t表示将编号为1,6的原子首尾相连形成环
c_pi 1,2,3,4,5,6    # 将编号为1,2,3,4,5,6的原子用大π键顺序相连
c 6,7           # 将编号为6,7的原子用单键相连
c 7,8 2         # 将编号为7,8的原子用双键相连
s               # 保存,此时分子test为苯甲醛
rn  苯甲醛      # 将分子test改名为苯甲醛
stl             # 将当前分子保存到本地
nm test1        
load molecules/苯甲醛.txt   # 加载保存过的分子至当前分子 molecules/为默认保存路径
== test1 苯甲醛 # 判断两分子是否相等
d 1             # 删除当前分子中编号为1的原子
s
q               # 退出
```
